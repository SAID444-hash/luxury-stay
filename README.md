# Luxury Stay Hotel Website

## Project Description
Luxury Stay Hotel Website is a simple responsive website built using HTML, CSS, and JavaScript.  
The website showcases hotel rooms, provides information about the hotel, and allows users to contact the hotel through a contact form.

This project demonstrates basic front-end web development skills including layout design, responsive styling, and simple interactivity.

---

## Features
- Responsive navigation bar
- Hero section with hotel background image
- Rooms section displaying different hotel rooms
- About section describing the hotel
- Contact form for customer inquiries
- "Book Now" button with booking confirmation message
- Smooth scrolling navigation

---

## Technologies Used
- HTML5
- CSS3
- JavaScript
